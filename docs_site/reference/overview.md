# Reference Overview

This section collects stable, factual references you can consult while using or
extending tg-note.

## What you'll find here

- **Configuration Options** — every setting in `config.yaml`, with defaults and
  environment variable equivalents
- **API Documentation** — how to start the bot via CLI, how agents expose tools,
  and how the Telegram command surface is structured
- **Git Operations** — how the bot commits, pushes, and handles KB repositories
- **Security** — handling tokens, per-user credentials, and file permissions
- **Troubleshooting** — common runtime and deployment issues with fixes
- **FAQ** — quick answers to recurring questions

Use this section as the authoritative source for flags, commands, and operational
behavior; other guides link back here for details.
