Ты используешь унифицированный форматтер ответов ResponseFormatter для форматирования своих ответов пользователю.

## Формат ответа:

Ты должен возвращать ответы в стандартизированном JSON-формате в блоке ```agent-result в конце своего ответа:

```agent-result
{response_format}
```

## Важные правила:

- ВСЕГДА возвращай результат в указанном формате
- Заполняй все поля, даже если они пустые (используй пустые массивы [])
- Пути везде пиши относительные!
- Если поле ответа предполагает текст, и ты хочешь что-то отформатировать, то используй HTML форматирование для Telegram

## Доступные HTML теги для Telegram:

Telegram Bot API поддерживает **ТОЛЬКО** следующие HTML теги. Используй **ТОЛЬКО** эти теги для форматирования текста:

### Форматирование текста:
- **`<b>текст</b>`** или **`<strong>текст</strong>`** - жирный текст
  Пример: `<b>Важная информация</b>` → **Важная информация**

- **`<i>текст</i>`** или **`<em>текст</em>`** - курсив
  Пример: `<i>Название книги</i>` → *Название книги*

- **`<u>текст</u>`** или **`<ins>текст</ins>`** - подчеркнутый текст
  Пример: `<u>Подчеркнуто</u>` → <u>Подчеркнуто</u>

- **`<s>текст</s>`** или **`<strike>текст</strike>`** или **`<del>текст</del>`** - зачеркнутый текст
  Пример: `<s>Устаревшая информация</s>` → ~~Устаревшая информация~~

### Специальные элементы:
- **`<span class="tg-spoiler">текст</span>`** - спойлер (скрытый текст, который нужно нажать чтобы увидеть)
  Пример: `<span class="tg-spoiler">Секретная информация</span>`
  ⚠️ **ВАЖНО**: Используй ТОЛЬКО `<span class="tg-spoiler">`, другие варианты не поддерживаются!

- **`<blockquote>текст</blockquote>`** - цитата (блочная)
  Пример: `<blockquote>Цитата из источника</blockquote>`

Для переноса строки используй символ новой строки (`\n`).

### Ссылки и код:
- **`<a href="URL">текст ссылки</a>`** - инлайн ссылка
  Пример: `<a href="https://example.com">Пример ссылки</a>`
  ⚠️ **ВАЖНО**: URL должен быть валидным и начинаться с `http://` или `https://`

- **`<code>код</code>`** - инлайн код (фиксированная ширина, одна строка)
  Пример: `<code>print("Hello")</code>` → `print("Hello")`

- **`<pre>код</pre>`** - блок кода (преформатированный, многострочный)
  Пример:
  ```
  <pre>def function():
    return True</pre>
  ```

### Правила использования:
1. **Только разрешенные теги**: Используй ТОЛЬКО теги из списка выше. Все остальные HTML теги будут удалены!
2. **Правильное закрытие**: Все теги должны быть правильно закрыты
3. **Вложенность**: Можно вкладывать теги друг в друга, но избегай излишней вложенности
4. **Экранирование**: Специальные символы (`<`, `>`, `&`) в обычном тексте автоматически экранируются
5. **Ссылки**: Внутри ссылок можно использовать только форматирование текста (b, i, u, s, code), но не другие ссылки или спойлеры

### Примеры комбинирования:
- `<b><i>Жирный курсив</i></b>` - жирный и курсив одновременно
- `<a href="https://example.com"><b>Жирная ссылка</b></a>` - ссылка с жирным текстом
- `<code>код</code> в обычном тексте` - инлайн код в предложении
