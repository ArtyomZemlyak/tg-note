# Работа с изображениями и элементами из PDF

**ВАЖНО:** Если во "Входящей информации" упоминаются сохраненные изображения (например, "сохранено как: images/img_123456.jpg") или извлеченные элементы из PDF (изображения, таблицы), ты МОЖЕШЬ и ДОЛЖЕН использовать их в создаваемых markdown файлах!

## Как вставлять изображения в markdown:

1. **Используй относительные пути** от текущего файла к папке images:
   - Если создаешь файл в корне KB: `![Описание](images/img_123456.jpg)`
   - Если создаешь файл в topics/: `![Описание](../images/img_123456.jpg)`
   - Если создаешь файл в topics/subfolder/: `![Описание](../../images/img_123456.jpg)`

2. **Добавляй осмысленные описания** (alt text) для изображений:
   - Плохо: `![image](images/img_123456.jpg)`
   - Хорошо: `![График роста выручки компании за 2023 год](images/img_123456.jpg)`
   - Отлично: `![График роста выручки компании: +45% в Q4 2023](images/img_123456.jpg)`

3. **Размещай изображения логично**:
   - Рядом с текстом, который они иллюстрируют
   - После заголовка соответствующего раздела
   - Не в начале файла (сначала заголовок и вводный текст)

4. **Используй все изображения из входных данных**:
   - Если пришло изображение, оно было прислано не просто так
   - Найди ему место в создаваемых заметках
   - Если изображение содержит текст (из OCR), ты можешь использовать его для контекста

## Работа с извлеченными элементами из PDF:

Когда обрабатывается PDF документ (например, из arxiv), система автоматически извлекает:
- **Изображения** (графики, диаграммы, рисунки) - сохраняются в `images/`
- **Таблицы** - сохраняются в `tables/` в формате JSON

### Извлеченные изображения из PDF:

Если во "Входящей информации" есть раздел "Извлеченные изображения из PDF", это означает, что из PDF были извлечены изображения. Ты ДОЛЖЕН:

1. **Использовать все извлеченные изображения** в создаваемом markdown файле
2. **Правильно указывать пути**:
   - Из `topics/`: `../images/pdf_filename_timestamp_0.png`
   - Из `topics/subfolder/`: `../../images/pdf_filename_timestamp_0.png`
3. **Добавлять описания** на основе контекста PDF:
   - Если это график из научной статьи, опиши что он показывает
   - Если это диаграмма, объясни её назначение
   - Используй текст из PDF для понимания контекста изображения

### Извлеченные таблицы из PDF:

Если во "Входящей информации" есть раздел "Извлеченные таблицы из PDF", это означает, что из PDF были извлечены таблицы. Ты МОЖЕШЬ:

1. **Воссоздать таблицу в markdown** используя данные из JSON файла
2. **Ссылаться на таблицу** если она слишком большая: `[См. таблицу](../tables/table_filename.json)`
3. **Использовать данные таблицы** в тексте заметки

**Пример использования извлеченных элементов:**

```markdown
# Анализ статьи из arxiv

## Основные результаты

![График точности модели на тестовом наборе данных](../images/pdf_arxiv_1705334567_0.png)

Как видно из графика выше, модель показывает точность 95%...

## Сравнительная таблица

| Метод | Точность | Время обучения |
|-------|----------|---------------|
| Baseline | 85% | 2 часа |
| Наш метод | 95% | 3 часа |

(Данные из [таблицы](../tables/table_arxiv_1705334567_0.json))
```
